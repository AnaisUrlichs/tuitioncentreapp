This is a simple Java project written in BlueJ. It is responsible for tracking 
- the students who are part of different classes, 
- the ratings of classes,
- the books required by each student,
- the total revenue collected for the classes,
- the ratings of each class by the students,

## Fastest way on running the project

* Download BlueJ
* Clone this git repository
* Open the repository in BlueJ
* Select the LoanSystem Class and "Compile" -- by right clicking on the class
* Run the main() method
